# AX 표준 코드체계 v1.0

## 1. 5단계 과업 코드체계 (A~E)

| 대분류 | 코드 | 항목명 | 설명 |
|--------|------|--------|------|
| **A. 기획·전략** | A01 | 사업기획 | 사업 타당성 분석, 추진전략 수립 |
| | A02 | 요구사항 분석 | 이해관계자 요구사항 도출 |
| | A03 | 벤치마킹 | 유사 사례 조사 및 분석 |
| | A04 | 로드맵 수립 | 단계별 추진계획 수립 |
| **B. 설계·모델링** | B01 | 서비스 설계 | 서비스 청사진, 사용자 여정 |
| | B02 | 데이터 모델링 | ERD, 데이터 흐름도 |
| | B03 | 프로세스 설계 | BPM, 업무 프로세스 정의 |
| | B04 | UI/UX 설계 | 화면설계, 프로토타입 |
| | B05 | 아키텍처 설계 | 시스템 구조, 인프라 설계 |
| **C. 개발·구축** | C01 | 프론트엔드 개발 | 웹/앱 UI 구현 |
| | C02 | 백엔드 개발 | API, 서버 로직 구현 |
| | C03 | 데이터 파이프라인 | ETL, 데이터 수집·처리 |
| | C04 | AI/ML 모델 개발 | 모델 학습, 튜닝, 배포 |
| | C05 | 인프라 구축 | 클라우드, 온프레미스 환경 |
| | C06 | 통합·연동 | 외부 시스템 연계 |
| **D. 운영·지원** | D01 | 시스템 운영 | 모니터링, 장애대응 |
| | D02 | 데이터 운영 | 데이터 품질관리, 갱신 |
| | D03 | 사용자 교육 | 교육자료 제작, 워크숍 |
| | D04 | 기술지원 | 헬프데스크, 이슈대응 |
| **E. 관리·품질** | E01 | 프로젝트 관리 | 일정·범위·리스크 관리 |
| | E02 | 품질관리 | QA, 테스트 |
| | E03 | 보안·컴플라이언스 | 보안점검, 규제준수 |
| | E04 | 문서화 | 산출물 작성, 인수인계 |

**총 24개 세부 코드**

## 2. 인력등급 체계 (S~D)

| 등급 | 명칭 | 경력 | 단가배율 | 역할 예시 |
|------|------|------|----------|-----------|
| **S** | 특급기술자 | 15년+ | 1.5 | 총괄PM, 수석아키텍트 |
| **A** | 고급기술자 | 10~15년 | 1.2 | 책임PM, 리드개발자 |
| **B** | 중급기술자 | 5~10년 | 1.0 (기준) | 선임개발자, 분석가 |
| **C** | 초급기술자 | 2~5년 | 0.8 | 개발자, 디자이너 |
| **D** | 신입기술자 | 0~2년 | 0.6 | 주니어, 인턴 |

## 3. 패키지 유형 (4종)

| 패키지 | 대상 | 포함 범위 | 표준 기간 | 가격대 |
|--------|------|-----------|-----------|--------|
| **Lite** | 소규모·검증 | A01-A02, B01-B02, C01-C02 | 1~2개월 | 1,000~3,000만원 |
| **Standard** | 일반 프로젝트 | A~C 전체, D01-D02 | 3~6개월 | 3,000~8,000만원 |
| **Pro** | 대규모·복합 | A~E 전체 | 6~12개월 | 8,000만원~2억원 |
| **Enterprise** | 공공·대기업 | 전체 + 보안·컴플라이언스 강화 | 12개월+ | 2억원+ |

## 4. 배율 체계

### 4.1 난이도 배율
| 구분 | 배율 | 기준 |
|------|------|------|
| 표준 | 1.0 | 일반적인 요구사항 |
| 복잡 | 1.3 | 다중 연동, 고성능 요구 |
| 고난도 | 1.6 | 신기술 적용, 규제 대응 |

### 4.2 긴급도 배율
| 구분 | 배율 | 기준 |
|------|------|------|
| 일반 | 1.0 | 표준 일정 |
| 급행 | 1.2 | 50% 단축 |
| 특급 | 1.5 | 70%+ 단축 |

### 4.3 공공조달 배율
| 구분 | 배율 | 적용 |
|------|------|------|
| 민간 | 1.0 | 일반 기업 |
| 공공 | 0.85~0.95 | 조달청 단가 적용 |
| 수의계약 | 1.0 | 2,000만원 이하 직접계약 |

## 5. 산출 공식

```
총 금액 = Σ(과업코드별 공수 × 인력등급 단가 × 난이도배율 × 긴급도배율)
```

### 예시: 제주DA 프로젝트
- 과업: A01, A02, B02, B03, C03, C04, D02, E01
- 인력: S급 1명, A급 2명, B급 3명
- 기간: 6개월
- 난이도: 복잡(1.3)
- 긴급도: 일반(1.0)
