## 프로그램 개요

**INsights from INside**는 조직 내부 구성원의 인사이트를 통해 문제를 진단하고 해결방안을 도출하는 Self-Consulting 프로그램입니다.

| 구분 | 내용 |
|------|------|
| 진단 프레임 | McKinsey OHI 기반 조직건강 진단 |
| 진행 방식 | 설문 + 면담 + 워크숍 + 보고회 |
| 제안가 | 4,000,000원 (VAT 별도) |
| 불포함 | 리더십 강사료, 워크숍 다과/상품권 |

---

## 프로세스

1. **문서 분석**: 기존 내부 조사 자료 검토
2. **설문**: 무기명 자가기입식 조직건강 설문
3. **면담**: 구성원 8명 + 상무 (별도)
4. **원인 파악 워크숍**: 2회 (4명씩)
5. **해결방안 워크숍**: 전원 참석 또는 2회
6. **지표 공개 보고회**: 종합 결과 발표
7. **외부 강사 특강**: 구성원이 선택한 강사 초빙 (별도)
