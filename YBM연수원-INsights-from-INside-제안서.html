1. Program Overview

1.1 INsights from INside

“The answers are already INside.”

INsights from INside는 조직 내부 구성원의 인사이트(INsights)를 통해 문제를 진단하고 해결방안을 도출하는 Self-Consulting Program입니다.

Framework McKinsey OHI 기반 조직건강 진단
Method 퍼실리테이션 기반 자기 진단
Philosophy 해답은 이미 내부에 있다

1.2 Why INsights from INside

기존 접근 한계
리더십 교육 일방향 전달, 현장 적용 어려움
팀빌딩 워크숍 일회성 이벤트, 지속성 부재
외부 컨설팅 현장 맥락 부재, 수용성 낮음
설문조사 정량 데이터만, 맥락 파악 어려움

INsights from INside는 다릅니다.

  • 정량(설문) + 정성(면담) 병행
  • 구성원이 직접 원인을 규명하고 방안을 도출
  • 중립적 퍼실리테이션으로 솔직한 대화 유도
  • 합의된 지표와 실행과제 도출

4. Methodology

적용 기법

기법 활용 단계 목적
Framed 3Why 원인 파악 표면적 → 잠재적 → 근본적 원인 계층화
Customized Fishbone 원인 파악 원인 구조화 및 시각화
Customized Brainwriting 전 과정 익명 의견 수렴, 의견 쏠림 방지
