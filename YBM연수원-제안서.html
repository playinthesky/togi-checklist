1. Program Overview

1.1 INsights from INside

"The answers are already INside."

INsights from INside는 조직 내부 구성원의 인사이트(INsights)를 통해 문제를 진단하고 해결방안을 도출하는 Self-Consulting Program입니다.

FrameworkMcKinsey OHI 기반 조직건강 진단
Method퍼실리테이션 기반 자기 진단
Philosophy해답은 이미 내부에 있다

1.2 Why INsights from INside

기존 접근한계
리더십 교육일방향 전달, 현장 적용 어려움
팀빌딩 워크숍일회성 이벤트, 지속성 부재
외부 컨설팅현장 맥락 부재, 수용성 낮음
설문조사정량 데이터만, 맥락 파악 어려움

INsights from INside는 다릅니다.

  • 정량(설문) + 정성(면담) 병행
  • 구성원이 직접 원인을 규명하고 방안을 도출
  • 중립적 퍼실리테이션으로 솔직한 대화 유도
  • 합의된 지표와 실행과제 도출

3. Program Structure

3.1 Process Flow

[Phase 1] [Phase 2] [Phase 3] 진단 해석 정착 ───────── ───────── ───────── 문서 분석 원인 파악 워크숍 지표 공개 보고회 설문 (OHI) 해결방안 워크숍 외부 강사 특강 면담 3타임 강사 수렴

3.2 상세 일정
