## 3. 품목별 검수 기준

### 3.1 산출물 검수 기준

| 구분 | 검수 기준 |
|------|----------|
| 문서류 | 규격(페이지 수) 충족, 내용 완결성, 오탈자 없음 |
| 소스코드 | 정상 동작, 주석 포함, 테스트 통과 |
| ERD/설계서 | 표준 표기법 준수, 연관관계 명확 |

### 3.2 실물 검수 기준

| 구분 | 검수 기준 |
|------|----------|
| 인쇄물 | 규격 충족, 인쇄 품질 양호, 오탈자 없음 |
| 현수막/배너 | 규격 충족, 출력 품질 양호, 색상 정확 |
| 영상 | 해상도 충족, 자막 포함, 음질 양호 |

### 3.3 용역 검수 기준

| 구분 | 검수 기준 |
|------|----------|
| 파실리테이터 | 자격증 또는 경력증빙, 원탁회의 정상 진행 |
| 기록자 | 기록물 품질, 실시간 기록 |
| 현장세팅 | 시간 내 설치/철거 완료, 안전사고 없음 |

## 5. 품목 코드 체계

```
[대분류][소분류]-[순번]
예: F04-01 = F(실물용역) + 04(파실리테이터) + 01(첫번째 품목)

대분류:
A = 기획·전략
B = 설계·모델링
C = 개발·구축
D = 운영·지원
E = 관리·품질
F = 실물·용역
```
