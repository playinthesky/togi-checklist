## 1. Program Overview

### 1.1 INsights from INside

> *"The answers are already INside."*

INsights from INside는 조직 내부 구성원의 인사이트(INsights)를 통해 문제를 진단하고 해결방안을 도출하는 **Self-Consulting Program**입니다.

| | |
|---|---|
| **Framework** | McKinsey OHI 기반 조직건강 진단 |
| **Method** | 퍼실리테이션 기반 자기 진단 |
| **Philosophy** | 해답은 이미 내부에 있다 |

### 1.2 Why INsights from INside

| 기존 접근 | 한계 |
|-----------|------|
| 리더십 교육 | 일방향 전달, 현장 적용 어려움 |
| 팀빌딩 워크숍 | 일회성 이벤트, 지속성 부재 |
| 외부 컨설팅 | 현장 맥락 부재, 수용성 낮음 |
| 설문조사 | 정량 데이터만, 맥락 파악 어려움 |

**INsights from INside는 다릅니다.**

- 정량(설문) + 정성(면담) 병행
- 구성원이 직접 원인을 규명하고 방안을 도출
- 중립적 퍼실리테이션으로 솔직한 대화 유도
- 합의된 지표와 실행과제 도출

## 3. Program Structure

### 3.1 Process Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   [Phase 1]           [Phase 2]           [Phase 3]            │
│    진단                 해석                 정착               │
│   ─────────           ─────────           ─────────            │
│   문서 분석            원인 파악 워크숍      지표 공개 보고회     │
│   설문 (OHI)          해결방안 워크숍       외부 강사 특강       │
│   면담 3타임          강사 수렴                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 상세 일정

| Phase | 활동 | 내용 | 일정 |
|-------|------|------|------|
| **사전** | 문서 분석 | 기존 내부 조사 자료 검토 | - |
| | 설문 배포 | 무기명 자가기입식, OHI 기반 | - |
| **Day 1** | 면담 | 구성원 8명 (3타임) | 1일 |
| | | 상무 별도 면담 | |
| **Day 2** | 원인 파악 워크숍 | 4명 × 2회 (각 2h) | 1일 |
| **Day 3** | 해결방안 워크숍 | 전원 또는 4명 × 2회 | 1일 |
| | | 희망 강사 수렴 | |
| **Day 4** | 지표 공개 보고회 | 종합 결과 발표 | - |
| **Day 5** | 외부 강사 특강 | 구성원 선택 강사 | 별도 |

### 3.3 면담 설계

**구성원 면담 (8명)**

| 타임 | 시간 | 대상 |
|------|------|------|
| 1타임 | 10:00~11:30 | 3인 |
| 2타임 | 13:30~15:00 | 3인 |
| 3타임 | 15:30~17:00 | 2인 |

**상무 면담**: 구성원 면담 종합 후 별도 진행

### 3.4 워크숍 설계

**원인 파악 워크숍**

| 회차 | 시간 | 대상 | 목적 |
|------|------|------|------|
| 1회 | 10:00~12:00 | 4명 | 면담 쟁점 → 원인 규명 |
| 2회 | 14:00~16:00 | 4명 | 면담 쟁점 → 원인 규명 |

**해결방안 워크숍**

| 옵션 | 구성 | 대상 |
|------|------|------|
| A안 | 1회 × 3h | 8명 전원 |
| B안 | 2회 × 2h | 4명씩 |

## 4. Methodology

### 적용 기법

| 기법 | 활용 단계 | 목적 |
|------|----------|------|
| **Framed 3Why** | 원인 파악 | 표면적 → 잠재적 → 근본적 원인 계층화 |
| **Customized Fishbone** | 원인 파악 | 원인 구조화 및 시각화 |
| **Customized Brainwriting** | 전 과정 | 익명 의견 수렴, 의견 쏠림 방지 |
| **Customized Matrix** | 해결방안 | 심각성 × 선호도 우선순위화 |

### Framed 3Why

문제를 해결하고자 하는 범위 내에서:

```
[표면적 원인] → [잠재적 원인] → [근본적 원인]
     ↓              ↓              ↓
  현상 수준        구조 수준        문화 수준
```
