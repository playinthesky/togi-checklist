## 1. 사업 개요

### 1.1 추진 배경

제주특별자치도 농업기술원은 농산물 수급안정을 위한 '제주DA(Data Agriculture)' 플랫폼을 운영하고 있습니다. 2025년 파트너스 발대(200명) 이후 3회의 원탁회의를 통해 4종의 개선사항을 도출·반영한 바 있으며, 2026년에는 본격적인 사용자 참여형 개선 체계를 구축하고자 합니다.

### 1.2 사업 목적

| 목적 | 내용 |
|------|------|
| **사용성 개선** | 고령·여성 농업인 친화적 UI/UX 개선안 도출 |
| **현장 검증** | 실사용 환경에서의 기능 오류 및 불편사항 발굴 |
| **참여 확대** | 농업인 주도의 플랫폼 거버넌스 기반 마련 |
| **지속 환류** | 개발사 연계를 통한 실질적 개선 반영 |

### 1.3 사업 개요

| 항목 | 내용 |
|------|------|
| **사업명** | 제주DA 파트너스 운영 |
| **사업기간** | 2026. 6. ~ 12. (7개월) |
| **총 사업비** | 50,000천원 (VAT 포함) |
| **원탁회의** | 8회 × 25인 (총 200명) |
| **대상 작물** | 노지감귤, 월동무, 양배추, 당근, 마늘, 양파 |

## 4. 원탁회의 운영

### 4.1 표준 타임테이블 (2시간)

| 시각 | 소요 | 단계 | 활동 |
|------|------|------|------|
| 11:30~12:30 | 60분 | 사전 세팅 | 테이블 배치, AV 점검, 준비물 세팅 |
| 12:30~13:00 | 30분 | 입장·접수 | 명찰 배부, 아이스브레이크 |
| 13:00~13:30 | 30분 | 점심 식사 | 라포 형성 |
| 13:30~13:40 | 10분 | 개회 | 환영, 목적 공유, 그라운드룰 |
| 13:40~13:55 | 15분 | 앱 현황 | 제주DA 기능 시연, 사용현황 공유 |
| 13:55~14:15 | 20분 | 개별 기록 | 사일런트 브레인스토밍, 포스트잇 |
| 14:15~14:45 | 30분 | 모둠 토론① | 이슈 분류, 그룹핑, 원인 진단 |
| 14:45~14:50 | 5분 | 휴식 | |
| 14:50~15:15 | 25분 | 모둠 토론② | 개선 아이디어 도출 (HMW) |
| 15:15~15:25 | 10분 | 전체 공유 | 모둠별 발표, 점 투표 |
| 15:25~15:30 | 5분 | 마무리 | 만족도 설문, 후속 안내 |

### 4.2 운영팀 구성 (6인 체제)

| 역할 | 인원 | 담당 업무 |
|------|------|-----------|
| 기획자 | 1 | 사업 총괄, 클라이언트 소통 |
| 리딩 퍼실리테이터 | 1 | 전체 진행, 시간 관리 |
| 현장보조 | 2 | 접수, 물품, 진행 지원 |
| 기술오퍼레이터 | 1 | 앱 시연, AV 관리 |
| 분석팀 | 1 | 현장 기록, 포스트잇 디지털화 |

### 4.3 진행 원칙

1. **사일런트 브레인스토밍 우선** - 의견 쏠림 방지
2. **이슈 → 원인 → 개선안** 순서로 논리 단계화
3. **점 투표(dot voting)** 로 정량적 우선순위 도출
4. **퍼실리테이터는 내용 불개입** - 진행 구조만 관리
5. **전 과정 기록** - 사진, 녹음, 포스트잇 디지털화
